The Scholar’s Oath, a 2D educational Metroid Vania game that integrates gamified learning techniques with interactive exploration.
The project seeks to address declining engagement and motivation in analytical disciplines such as mathematics and programming by transforming learning into a creative, game-based experience.

The project is developed using frontend (Unity) and backend (ASP.NET API with PostgreSQL), managed within a single repository for ease of deployment and evaluation. The game replaces traditional combat with quiz-based encounters that challenge players intellectually rather than physically. Through problemsolving and exploration, players reinforce cognitive understanding and achieve mastery of academic topics.
